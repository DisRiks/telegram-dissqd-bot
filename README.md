# Telegram Shop Bot

Простой Telegram-бот для продажи услуг через меню.

Что умеет бот:

- показывает список услуг;
- ведет пользователя на FunPay по кнопке;
- отправляет бесплатную сборку файлом или ссылкой;
- работает через обычное меню Telegram.

## Быстрый старт

1. Создайте `.env` на основе `.env.example`.
2. Укажите `BOT_TOKEN`.
3. При необходимости положите бесплатную сборку в `free/build.zip`.
4. Запустите:

```powershell
.venv\Scripts\python.exe bot.py
```

## Где менять товары

Каталог редактируется в файле [catalog.py](C:\Users\123\Desktop\telegram-shop-bot\catalog.py).

Для каждой услуги можно поменять:

- название;
- описание;
- ссылку на FunPay.

## Бесплатная сборка

Есть два способа выдачи:

- бот отправляет локальный файл `FREE_BUILD_PATH`;
- бот показывает ссылку из `FREE_BUILD_URL`.

Если не указать ни файл, ни ссылку, бот подскажет, что нужно настроить.
